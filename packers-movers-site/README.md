# ShiftEase Packers & Movers Website

A clean, modern and fully responsive single-page website for a Packers & Movers business. Designed to convert visitors into leads with prominent quote forms, WhatsApp chat, trust signals, and clear service offerings.

## Features

- 🎨 Modern, clean design with professional blue/orange palette
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Lightning fast — pure HTML/CSS/JS, zero dependencies
- 📝 Multiple lead-capture forms (instant quote + contact)
- 💬 Floating WhatsApp chat button
- ❓ Interactive FAQ accordion
- ⭐ Customer testimonials section
- 🚚 Service grid covering all major moving categories
- 🔍 SEO-optimized meta tags
- ♿ Accessible markup and smooth scroll

## Deploy to Vercel

This site is a static site — just upload the `index.html`, `styles.css` and `script.js` files (or the whole folder) to Vercel. No build step required.

## Files

- `index.html` — main page
- `styles.css` — all styling
- `script.js` — interactivity (mobile menu, smooth scroll, reveal animations)
- `vercel.json` — Vercel routing config
